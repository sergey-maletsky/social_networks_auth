Версия Java: 1.8.0_131

Инструкция по развертыванию:

I Общая информация

1. приложение доступно на порту 9090
2. Swagger Api документация доступна по адресу http://localhost:6060/swagger-ui.html

II Подготовка базы данных

Настройки подключения базы данных находятся в файле src/main/resources/spring/database.properties

1. Установить PostgreSQL 9.6
2. Выполнить createdb.sql скрипт(в корне проекта) из PostgreSQL консоли под супер пользователем:
psql -U postgres -a -f createdb.sql

Скрипт создаст базу данных "snauth" и нового пользователя "snm" с правами для доступа к этой бд.

III Инструменты для сборки проекта

В проекте используются Gradle сборщик проектов.
Для того, чтобы собрать и развернуть проект, необходимо:
1. установить gradle, если еще не установлен.
    1) перейти по ссылке https://services.gradle.org/distributions/gradle-4.9-bin.zip
    чтобы скачать архив с gradle
    2) распаковать в необходимую директорию
    3) установить в переменную окружения PATH(Linux) или Path(Win) путь <gradle-path>/bin
2. перейти в корень проекта
3. выполнить из терминала gradle wrapper
4. затем выполнить ./gradlew clean build(Linux) или gradlew.bat clean build(Win)
5. и затем выполнить java -jar build/libs/social_networks_auth-1.0.0.jar
6. в браузере открыть http://localhost:9090/

IV Работа с приложением

На основной странице доступны ссылки для регистрации/авторизации через социальные сети,
которые подписаны соответственно.
Пример: мы решили войти через vk.com аккаунт.
Нам потребуется:
1. иметь аккаунт в данной соц. сети или зарегистрироваться в vk.com(в случае отсутствия аккаунта,
вам будет предложено зарегистрироваться).
2. при доступном аккаунте сначала необходимо зарегистрироваться в данной системе т.е.
кликнуть по ссылке VKontakte под словами Регистрация через:.
3. появятся слова об успешной регистрации, после чего перейти по ссылке Вернуться на главную страницу
4. кликнуть по ссылке VKontakte под словами Вход через:. Система поприветствует пользователя, вошедшего в нее
